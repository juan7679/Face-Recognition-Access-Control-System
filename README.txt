No hay como un requerimiento extra para este proyecto, solo es hacer una carpeta con el nombre con el que se reconocerá a la persona y poner ahí
fotos de la persona a reconocer, hacer el número de carpetas necesarias para las personas distintas a reconocer, poner todo esto en la carpeta "base_datos".
No requiere tiempo de entrenamiento y de preferencia, evitar que la cámara se encuentre a contraluz, esto aplica a las fotos.
Hay un conflicto de librerías entre este y el proyecto final, se debe ejecutar uno de estos en una burbuja o borrar las librerías usadas aquí.
